Проект "Пиццерия"

Проект содержит 5 страниц: 
1. welcome страница:
- общая информация о пиццерии;
- осуществляется ввод имени клиента, которая отображается в AppBar для больших экранов;
- seach поле для поиска информации по определенному заказу(номер заказа для ввода получаем при создании заказа);
2. страница меню: 
- получение меню с api, отображение на экране;
- функционал добавления пиццы в корзину;
- выбор количества заказываемых пицц определенного вида (увеличение или уменьшение их количества);
- удаление из корзины всех пицц определенного вида;
- отображение общего количества выбранных пицц и их общей стоимости;
- переход в корзину для заказа;
- навигация на основную страницу по клику на "Fast react Pizza Co." в AppBar
- отображение Loader, во время загрузки меню;
3. корзина клиента:
- отображаются добавленные в корзину пиццы, их количество и стоимость;
- возможность удалить полностью определенный тип пиццы из корзины;
- возможность уменьшить/увеличить количество определенного типа пиццы;
- возможность полностью очистить корзину;
- переход к оформлению заказа;
- кнопка возврата к меню;
4. страница создания заказа:
- ввод данных клиента (при вводе телефона нужно ввести как минимум 5 цифр);
- возможность получить геолокацию пользователя;
- возможность выбрать приоритетность для заказа(заказ выполнится быстрее);
- отображение полной стоимости заказа;
- сохранение заказа, навигация к деталям заказа, отображение Loader при сохранении;
5. детали заказа:
- отображаются детали заказа, статус;
- возможность поменять приоритетность заказа;
- получаем номер заказа, которое можем использовать чтобы посмотреть детали заказа из search поля в AppBar(вводит номер без #);
